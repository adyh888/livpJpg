# LIVP 图片提取工具

一个基于 uni-app 开发的 LIVP 文件图片提取工具，支持将 iPhone 拍摄的 LIVP 文件中的图片（包括 HEIC 格式）提取并转换为 JPG 格式，支持批量处理和下载。

## 功能特点

- 支持选择多个 LIVP 文件进行批量处理（最多 200 个文件）
- 自动提取 LIVP 文件中的 JPG 和 HEIC 图片
- 将 HEIC 格式图片自动转换为 JPG 格式
- **5 列紧凑预览布局**：预览图采用 5 列布局，节省资源，提高性能
- **点击放大预览**：点击任意预览图可全屏查看，支持左右滑动浏览所有图片
- **顺序排列**：预览图从左到右顺序排列，不会出现中间留空的情况
- 支持单张图片下载和批量打包下载
- **现代化 UI 设计**：采用渐变背景、毛玻璃效果等现代设计元素，美观大气

## 技术栈

- 框架：uni-app（支持 Vue 3）
- 依赖库：
  - `jszip`：用于解析 LIVP 文件（本质是 ZIP 压缩包）
  - `heic-to`：用于将 HEIC 格式图片转换为 JPG

## 使用方法

1. 点击"点击选择 .livp 文件"区域
2. 选择一个或多个 LIVP 文件（支持批量选择，最多 200 个）
3. 等待文件解析和转换完成（处理过程中会显示加载提示）
4. 预览提取出的图片（5 列紧凑布局，节省资源）
5. **点击任意预览图可全屏放大查看**，支持左右滑动浏览所有图片
6. 可选择单张下载（点击预览图下方的下载按钮）或点击"下载全部图片"打包下载为 ZIP 文件

## 注意事项

- 处理大量或大型 LIVP 文件时可能需要较长时间，请耐心等待
- HEIC 转换功能依赖浏览器环境支持，部分旧浏览器可能存在兼容性问题
- 批量下载功能会将所有图片打包为 ZIP 文件
- 预览图采用小尺寸显示以节省资源，点击图片可查看大图
